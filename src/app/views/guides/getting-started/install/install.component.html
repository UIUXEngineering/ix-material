Sharing code should be convenient and simple. This platform utilizes Google tools such as the @angular/cli to build the platform projects and documentation app. In other words, the same tools as you would build a production app.

As you develop in your production app, you may want to share a module, component, etc. It should take minimal effort to "drop" into this architecture, wire to the platform to publish to NPM, unit test, document, and share.
