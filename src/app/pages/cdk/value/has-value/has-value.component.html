hasValue is an opinionated value check based on the type of data of the value. Evaluation of Values: Value Type {{element.valueType}} JS {{element.js}} undefined / null {{element['undefined']}} empty {{element['empty']}} defined {{element.defined}} not empty {{element['nonEmpty']}}

1 Scalar values such as numbers and booleans are true if they are defined. The boolean value of false and the number 0 evaluate true because they are defined ( they "have value").

2 An Array is not considered to "have value" unless it contains data ( non-empty array ).

3 An Object is not considered to "have value" unless it has properties ( non-empty object ).

4 A String is not considered to "have value" unless it has at least one character ( non-empty string ).
