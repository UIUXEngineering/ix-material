Icons are created using Sketch and exported as a single svg file. It is best to build only one icon per Sketch file. Other tools are available to concatenate icon svg files into one file for loading by Material.

Save a sketch file in this directory . You may copy a file as well, but save as a different name.

See the Sketch Docs to learn how to use Sketch.

Use max dimensions of 24px x 24px. Add a border that is not visible to ensure dimensions when loading icons in your app. A good source for reference is downloading a material svg icon and import into your sketch file to see how it's built.

See the Sketch Docs to learn how to export svg code.
